# Flipkart-Automation
Flipkart Automation Framework

This project is an end-to-end automation framework designed to test core functionalities of the Flipkart web application. It uses Selenium WebDriver, Java, TestNG, Cucumber BDD, and is fully Dockerized for scalable and environment-independent execution.
